* Disquss ve Resim Optimize

Settings.py üzerinden bu değerleri değiştirebilirsiniz.

    #Resim kalitesi
    IMAGE_QUALITY = 45
    # Küçük resim boyutu
    THUMBNAIL_SIZE=(300,300)
    
    #Api Key
    DISQUS_API_KEY = 'ApiKey'
    #Subdomain
    DISQUS_WEBSITE_SHORTNAME = 'aliyaman'
